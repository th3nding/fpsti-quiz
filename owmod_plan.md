# OWMod 地图工坊数据爬取方案

## 任务概述
爬取 https://www.owmod.net 从 2020 年到 2025 年的 100 个地图工坊信息，按热度排序，采用分层抽样。

## 网站结构分析
- 列表页：https://www.owmod.net/work/list/all?page={page}&sort=hot
- 详情页：https://www.owmod.net/work/view/{id}
- 共 144 页，每页约 12 个作品

## 爬取步骤

### 1. 爬取所有作品列表
需要访问 144 页获取所有作品的基本信息

### 2. 筛选 2020-2025 年数据
根据发布日期筛选

### 3. 分层抽样
- 高热度：前 10%，抽取 33 个
- 中热度：10%-90%，抽取 33 个
- 低热度：后 10%，抽取 34 个

### 4. 字段说明
- 作品 ID
- 名称
- 作者
- 发布日期
- 类型（娱乐/竞技/叙事/技能训练）
- 播放量
- 点赞数
- 当前状态
- 热度
- 代码

## 执行方案

由于无法直接运行 Python requests，我将使用浏览器逐个页面访问并记录数据。

### 方法 1：使用浏览器批量访问
1. 访问列表页，提取作品信息
2. 翻页继续爬取
3. 整理数据到 Excel

### 方法 2：使用 Playwright
如果浏览器工具支持，可以使用 Playwright 自动化爬取
